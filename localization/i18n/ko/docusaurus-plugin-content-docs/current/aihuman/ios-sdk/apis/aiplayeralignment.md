# AIPlayerAlignment

## AIPlayerVerticalAlignment

AIPlayer의 크기를 변경 할 때 수직 기준점을 정한다.

| Name                   | Description                                         |
| ----- | --------- |
| `top`                  | 상단 기준점   |
| `center`               | 중앙 기준점   |
| `bottom`               | 하단 기준점   |


## AIPlayerHorizontalAlignment

AIPlayer의 크기를 변경 할 때 수평 기준점을 정한다.

| Name                   | Description     |
| ----- | --------- |
| `left`                 | 왼쪽 기준점       |
| `center`               | 중앙 기준점       |
| `right`                | 오른쪽 기준점      |
