# AIGesture

| Modifier and Type | Method and Description |
| ----------------- | ---------------------- |
| `String`          | `getName()` 설정된 제스처의 이름을 가져온다.        |
| `ClipType`        | `getClipType()` 설정된 제스처 클립 타입을 가져온다. |
| `boolean`         | `isEnableSpeech()` 발화가 가능한 제스처인지 확인한다.    |
