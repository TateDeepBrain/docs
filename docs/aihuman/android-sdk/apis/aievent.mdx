# AIEvent

| Modifier and Type | Method and Description |
| ----------------- | ---------------------- |
| `AIEvent.Type`   |  `type` event's type. The values are <br/>- RES_LOAD_STARTED, <br/>- RES_LOAD_COMPLETED, <br/>- AICLIPSET_PLAY_PREPARE_STARTED, <br/>- AICLIPSET_PLAY_PREPARE_COMPLETED, <br/>- AICLIPSET_PRELOAD_STARTED, <br/>- AICLIPSET_PRELOAD_COMPLETED, <br/>- AICLIPSET_PRELOAD_FAILED, <br/>- AICLIPSET_PLAY_STARTED, <br/>- AICLIPSET_PLAY_COMPLETED, <br/>- AICLIPSET_PLAY_FAILED, <br/>- AI_CONNECTED, <br/>- AI_DISCONNECTED, <br/>- AICLIPSET_PLAY_BUFFERING, <br/>- AICLIPSET_RESTART_FROM_BUFFERING, <br/>- AIPLAYER_STATE_CHANGED |
| `String`   |  `getSpeechText()` speech to play  |
| `String`   |  `getGesture()` gesture to play  |
